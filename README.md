## Overview
This project evaluates the effectiveness of retail promotions using Python to assess their impact on **sales performance**, **revenue**, and **customer purchasing behavior**.  
The analysis reflects a real-world Data Analyst workflow, focusing on transforming raw data into actionable business insights.


## Business Objective
Retail organizations invest significantly in promotional campaigns.  
This project aims to determine whether promotions drive measurable value and to identify patterns that can support data-driven decision-making.


## Technical Skills & Tools
- Python (Pandas, NumPy)
- Exploratory Data Analysis (EDA)
- Data Visualization (Matplotlib, Seaborn)
- Business KPI Analysis
- Insight Communication


## Key Insights
- Promotional periods show higher sales volume compared to non-promotional periods
- Certain products and categories demonstrate stronger responsiveness to discounts
- Promotions improve short-term revenue, emphasizing the importance of strategic targeting
